# Product Hunt Asset Philosophy: Terminal Cartography

Atomic's launch assets should feel like a precise map of autonomous engineering work. The visual system is dark, compact, and technical, with Catppuccin Mocha surfaces carrying most of the weight and blue, teal, mauve, green, yellow, and red acting as semantic signals. The work should feel meticulously crafted, not decorated: every line, pane, gate, and label exists to explain control, visibility, and safe execution.

The core form language is terminal cartography: panes, session graphs, code columns, review gates, and isolated containers arranged as navigable infrastructure. Space is disciplined and asymmetric, with dense technical areas balanced by clean negative space. The compositions should look like they were labored over by someone who understands developer tools deeply, with painstaking attention to alignment, contrast, and legibility.

Typography should be sparse and direct. Headlines state the point; labels anchor the diagram; captions clarify skepticism without becoming paragraphs. Code is shown as real-looking TypeScript, not decorative pseudo-code. Text must read as part of the interface, using weight and placement rather than hype.

Color carries trust. Blue means orchestration, teal means execution, green means approved output, yellow means review, red means uncontrolled risk, and mauve marks human judgment. The background uses quiet technical texture so the foreground stays crisp. The final images should feel reliable, minimal, and powerful: a Product Hunt gallery that answers "why do I need this?" before the viewer asks.
